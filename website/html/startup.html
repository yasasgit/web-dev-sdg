Startup

Sri Lankan government has planned to giveaway startup funds for school leavers selecting the best 2000 business ideas with the support of leading private businesses. School leavers can apply for this opportunity free of charge if they think that their idea would become a great success. The main purpose of this project is to create new jobs even if these startups may not become 100% success. We were able to start new businesses with the help of this program last year. A 3% of new businesses became great success and they were able to create more than 20000 new jobs. You can submit a request for this startup program using this form.
